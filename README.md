# ASP.NET / VB 網頁程式設計教學範例

伺服網頁程式設計課程教學範例，使用 ASP.NET Web Forms + VB 語言，示範網頁伺服器控制項與程式邏輯。

## `Ch03VBasic/`：VB 語法基礎

涵蓋 If-Then-Else、For-Next 迴圈、Do-Loop（多種變化）、Select Case、陣列、字串轉換、副程式/函式、全域與區域變數等 VB 基礎語法教學範例。

## `CH04Ch05WebControl/`：ASP.NET 網頁控制項

涵蓋 Label、TextBox、Button（含 LinkButton／ImageButton）、CheckBox（List）、RadioButton（List）、ListBox、DropDownList、Calendar、AdRotator、Wizard、Table、Panel、MultiView 等常用伺服器控制項，並包含幾個綜合應用範例（飲料/茶葉訂購系統、自助訂餐系統、個人履歷、帳號密碼登入）。

## `WebSite20220406/`

TextBox 元件補充範例。

## 環境需求

Visual Studio（含 ASP.NET Web Forms 開發工作負載），開啟對應資料夾內的 `.sln` 專案檔即可執行。
